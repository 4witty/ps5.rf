# пс5.rf - информационный портал playstation 5

простой и адаптивный сайт про playstation 5. новости, игры, характеристики консолей и обновления системы. всё собрано в одном месте для русскоязычных игроков.

## о проекте

пс5.rf это независимый информационный ресурс. здесь нет сложной логики и серверов, только чистый html, css и javascript. сайт быстро грузится, корректно работает на телефонах и выглядит аккуратно. мы следим за анонсами sony, собираем технические данные и пишем о новых играх.

## структура файлов

index.html - главная страница
about.html - информация о проекте
console.html - сравнение моделей ps5
games.html - список ожидаемых игр
news.html - новости и обновления
specs.html - характеристики ps5 pro
style.css - основные стили
*.css - стили отдельных страниц
script.js - общая логика и анимации
console.js - интерактив для страницы консолей

## важно

сайт не связан с sony interactive entertainment. все названия playstation, ps5, dualsense и другие являются торговыми марками их владельцев. проект создан только в образовательных и информационных целях.
